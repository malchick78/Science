Programming Assignment A6 --- Prefix Calculator
Programming Assignment A6 — Prefix Calculator
Due 27 May

Objectives

  • Using Java classes
  • Learn prefix and infix forms of expressions
  • Using trees

Task: calculate prefix expressions using binary a binary tree

An arithmetic expression is in prefix form when operators are written before their operands. Here are some examples of prefix expressions, with spaces denoted by _ and the values they evaluate to:

ExpressionValue
12 12
+_2_51 53
*_5_7 35
*_+_16_4_+_3_1 80

An expression (such as 12) that begins with an integer is a prefix expression that evaluates to itself. Otherwise, an expression is a prefix expression if it begins with an operator and is followed by two prefix expressions. In this latter case, the value of the expression is recursively computed from the values of its constituent prefix sub-expressions.

Write a program that allows the user to enter prefix expressions in a text field. The program reads the expression, evaluates it, and displays converted expression into the infix form followed by the value, to the user. See examples below. Assume that the user enters expressions that use only positive integers and the four operators +, -, * and /.

Your program should internally convert the prefix expression into a binary tree before evaluating it.
